# 우선순위 큐 ( 힙 )
최대 힙 : 부모 노드의 키 값이 자식 노드의 키 값보다 항상 큰 이진 트리 
* 중복 가능
* 완전 이진 트리 
* 느슨한 정렬 : 이진 탐색 트리와 달리 부모 자식간 크기만 정해져 있음


# 힙 정렬
* 높이 logN ( 완전 이진 트리 )
* 
힙 정렬이 유용한 경우는 가장 큰 값이 몇개 필요할 때이다.
